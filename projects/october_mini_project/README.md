# Women Techsters October Mini Project

## Overview
This is the individual showcase page for the October mini project.

## How to Contribute
1. Add your profile card in `index.html`.
2. Place your profile image in `images/`.
3. Update your bio, track, and any links.
4. Commit your changes and push to your branch.

## Project Structure
- `index.html` → main page
- `css/style.css` → styling
- `js/script.js` → scripts
- `images/` → profile images
- `README.md` → project description
